D - Retour à la ligne:

- Nowrap: pas de retour a la ligne (par defaut)
- wrap: retour a la ligne

E - Alignement:

les éléments sont organisé soit horizontalement soit verticalement. Cela definit ce qu'on appelle l'axe principal
Il y a aussi un axe secondaire (cross axis):
-si vos éléments sont aligné horizontalement l'axe secondaire est verticale
-si vos éléments sont aligné verticalement l'axe secondaire est horizontale


Alignement sur l'axe principal:

Afin d'aligner nos éléments sur l'axe principal on utilise: justify-content qui a comme valeurs:
-flex-start: aligner au debut (par defaut)
-flex-end: aligner a la fin
-center: aligner au centre
-space-between: il y a de l'espace entre eux, s'etire sur tout le ligne
-space-around: les éléments sont etirés sur toute l'axe mais laisse un espace aux extremités
-space-evenly: les éléments sont reparti uniformement dans le conteneur de l'alignement le long de l'axe principal.

F - Répartir sur plusieures lignes:

Si vous avez plusieures lignes sur votre flex-box vous pouvez choisir comment celle-ci seront réparties, avec align-content. Cette propriété permet de gerer l'alignement des differentes lignes ou colonnes les uns par rapport aux autres. Cette propriété n'aura d'effet que si le conteneur flexible possède plusieures lignes (si l'axe principal est horizontal ou plusieures colonnes (si l'axe principale est verticale) et que si le flex-wrap) est present.
Les valeurs de cette propriété:
-> Flex-start: placé au debut
-> Flex-end: placé a la fin
-> Center: placé au centre
-> Space-between: les éléments sont séparés avec un espace entre eux
-> Space-around: les éléments sont séparés avec un espace entre eux avec un espace aux extremités
-> Space-evenly: les éléments sont reparti uniformement dans le conteneur de l'alignement le long de l'axe principal.
-> Stretch: les éléments s'étirent pour occuper tout l'espace de la ligne
